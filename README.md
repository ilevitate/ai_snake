# 🐍 贪吃蛇游戏

一个基于 HTML5 Canvas 的贪吃蛇游戏，支持鼠标/触摸控制，具有平滑的动画效果和美观的视觉设计。


## ✨ 特性

- 🎮 **流畅的控制体验** - 使用高斯曲线算法实现平滑转向
- 🖱️ **多种控制方式** - 支持鼠标移动和触摸屏操作
- 📱 **移动端适配** - 响应式布局，支持手机/平板游玩
- 🎨 **精美的视觉效果** - 渐变色彩、发光效果、圆润的蛇身设计
- 🏆 **本地记分系统** - 自动保存最高分
- ⚡ **渐进难度** - 随着得分增加，蛇的速度会逐渐提升

## 🚀 快速开始

### 本地运行

1. 克隆或下载本项目
2. 直接用浏览器打开 `index.html` 文件
3. 点击"开始游戏"按钮即可游玩

### 本地服务器方式（推荐）

使用 Python 启动本地服务器：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 🎯 游戏机制

### 核心算法

**高斯平滑转向**：蛇的转向使用高斯函数进行平滑处理，使小角度转向更灵敏，大角度转向更平滑。

```javascript
// 高斯函数
f(x) = e^(-x²/(2σ²))
```

### 难度递增

- 初始速度：5 像素/帧
- 每得 100 分（吃 10 个食物）速度增加 0.5
- 最高速度：10 像素/帧

### 计分规则

- 每吃一个食物：+10 分
- 最高分自动保存到浏览器本地存储

## 📝 自定义配置

### 修改画布尺寸

编辑 `index.html`：

```html
<canvas id="gameCanvas" width="800" height="450"></canvas>
```

### 修改初始速度

编辑 `js/snake.js`：

```javascript
this.speed = 5;  // 修改这个值
```

### 修改食物颜色

编辑 `js/food.js`：

```javascript
this.colors = ['#ff6b6b', '#feca57', '#ff9ff3', '#54a0ff', '#5f27cd'];
```
